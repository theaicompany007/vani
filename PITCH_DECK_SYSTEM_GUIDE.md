# Industry-Driven Pitch Deck Template System - User Guide

## üéØ Overview

The Industry-Driven Pitch Deck Template System automatically generates personalized pitch presentations for **any company** in **any industry**. The system uses:

1. **Industry Context** - Industry-specific pain points, challenges, and messaging templates
2. **VANI Personas** - AI personas tailored to each industry (e.g., "Digital Sales Officer" for FMCG, "Activation Concierge" for Banking)
3. **RAG Knowledge Base** - Case studies and solutions from your knowledge base
4. **Target Data** - Company and contact information

## üöÄ How to Use - Step by Step

### Step 1: Navigate to Pitch Presentation Tab

1. Open the VANI Command Center dashboard
2. Click on **"Pitch Presentation"** tab in the top navigation bar

### Step 2: Select Industry (Optional but Recommended)

1. In the top-right corner, you'll see an **Industry Selector** dropdown
2. Select the industry for your target company (e.g., "FMCG", "Food & Beverages", "Financial Services")
3. The system will automatically filter targets and load industry-specific context

**Supported Industries:**
- FMCG / Food & Beverages / Retail Distribution
- Financial Services / Banking / FinTech / Credit Cards
- Manufacturing / B2B / Exports / Conglomerates
- Logistics / Supply Chain
- Healthcare
- Education
- Real Estate
- Technology / SaaS / Information Technology

### Step 3: Select or Create a Target

**Option A: Select Existing Target**
1. In the **"Select Target"** dropdown, you'll see all targets (filtered by industry if selected)
2. Click the **"Refresh"** button if you don't see newly created targets
3. Select the target company you want to pitch

**Option B: Create New Target**
1. Go to **"Target Hit List"** tab
2. Click **"Add Target"** or **"Create Target"**
3. Fill in:
   - Company Name (e.g., "Parle Agro")
   - Contact Name (e.g., "Pradeep Gidwani" - CEO)
   - Industry (select from dropdown - this is important!)
   - Role (e.g., "CEO", "CDO", "CMO")
   - Email, LinkedIn, Phone (optional but recommended)
4. Click **"Save"**
5. Return to **"Pitch Presentation"** tab
6. Click **"Refresh"** to see your new target

### Step 4: Generate Pitch

1. Make sure a target is selected in the dropdown
2. Click the **"Generate AI Pitch Presentation"** button
3. Wait for generation (usually 10-30 seconds)
4. The pitch will appear in the preview section below

### Step 5: Review Generated Pitch

The generated pitch includes:

1. **Title** - Includes VANI Persona name (e.g., "Digital Sales Officer for Parle Agro")
2. **Problem Statement** - Industry-specific pain points
3. **Solution** - How VANI solves the problem using the appropriate persona
4. **Hit List** - Customized card for the target company
5. **Trojan Horse** - Low-risk pilot approach (e.g., "Give us 5,000 Dead Stores")

### Step 6: Export or Send Pitch

**Export Options:**
1. **PDF Export**: Click the download icon next to the pitch (if available)
2. **PowerPoint Export**: Click the PPT icon (if available)
3. **Copy/Share**: Copy the content manually

**Send Options:**
1. Click **"Email"** button to send via email
2. Click **"WhatsApp"** button to send via WhatsApp
3. Click **"LinkedIn"** button to send via LinkedIn

## üí° Tips & Best Practices

1. **Always Set Industry**: When creating targets, always select the correct industry. This ensures the right persona and context are used.

2. **Refresh After Creating Targets**: If you create a new target, click "Refresh" in the Pitch Presentation tab to see it immediately.

3. **Industry Selector**: Use the industry selector in the top-right to filter targets and ensure industry context is loaded.

4. **Review Before Sending**: Always review the generated pitch before sending. The AI generates industry-specific content, but you may want to customize it.

5. **Use Hit List**: The "Hit List" section is customized for each company - it highlights their specific pain points and VANI's solution.

6. **Trojan Horse Strategy**: The "Trojan Horse" section suggests a low-risk pilot approach. This is industry-specific and helps with initial buy-in.

## üõ†Ô∏è Troubleshooting

**Problem**: "No targets found"
- **Solution**: Create targets in "Target Hit List" tab first, then refresh

**Problem**: "No targets found for this industry"
- **Solution**: Either create targets with that industry, or select "All Industries" to see all targets

**Problem**: Pitch doesn't seem industry-specific
- **Solution**: Make sure the target has an industry assigned, or select an industry in the top-right selector

**Problem**: Can't see newly created target
- **Solution**: Click the "Refresh" button next to the target dropdown

## üìä What Happens Behind the Scenes

1. **Industry Detection**: System reads industry from target or current industry context
2. **Persona Mapping**: Maps industry to appropriate VANI Persona (e.g., FMCG ‚Üí Digital Sales Officer)
3. **Context Loading**: Loads industry-specific pain points, use cases, and value propositions
4. **RAG Query**: Queries knowledge base for relevant case studies and solutions
5. **AI Generation**: Uses OpenAI to generate personalized pitch with all context
6. **Structured Output**: Formats pitch into Title, Problem, Solution, Hit List, Trojan Horse
